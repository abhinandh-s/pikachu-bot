## How to Use Pikachu

For detailed guide and demo see this [page](how-to-use.md).

### ​1. Direct Messaging

​Simply start a chat with @cma_buddy_bot and request the paper you need.

### ​2. Inline Query

​You can trigger Pikachu from any chat or group on Telegram. Just type your query starting with the bot's username: `@cma_buddy_bot <query>`

**​Query Formats:**

​PYQ: `p[PaperNumber]-[Term]-pyq`

​Example: `p5-26j-pyq`

​MQP: `p[PaperNumber]-[Term]-mqp-[Set]`

​Example: `p5-26j-mqp-s1a`
